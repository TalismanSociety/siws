## Prerequisites

- @polkadot/api - `v10.10.1` or higher.
- @polkadot/extension-dapp - `0.46.5` or higher

## Installation

Run the following command in your project to install SIWS into your project.

<Tabs>
<TabItem label="npm">

```sh
npm install @talismn/siws
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install @talismn/siws
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add @talismn/siws
```

</TabItem>
</Tabs>
