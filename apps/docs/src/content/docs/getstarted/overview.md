SIWS is a JavaScript package that lets you seamlessly integrate "Sign-In with Substrate" functionality into your applications. By utilizing SIWS, you can enhance the user experience within you applications by providing a convenient and secure method for users to sign in with their Substrate wallet, streamlining the authentication process for your application.

## Key Features

1. **Seamless Integration**
   Easily integrate SIWS into your application with just a few lines of code, enabling users to sign in with their Substrate wallet.

2. **Secure Authentication**
   Utilize the robust security measures of Substrate wallets to ensure a secure and reliable authentication process for your users.

3. **Full Typescript Support**
   The `@talismn/siws` package is built with comprehensive TypeScript support.
